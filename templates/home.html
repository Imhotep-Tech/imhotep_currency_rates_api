Available Endpoints

GET
/latest_rates/{api_key}/{base_currency}

Returns the latest exchange rates for all supported currencies relative to the specified base currency. Rates are updated daily and cached for optimal performance.

Base URL: https://imhotepexchangeratesapi.pythonanywhere.com
Parameters
Name Type Required Description
api_key string Yes Your unique API key
base_currency string Yes Currency code (USD, EUR, GBP, etc.)
Response Format
{
  "meta": {
    "base_currency": "USD",
    "last_updated_at": "2024-10-24T18:00:00"
  },
  "data": {
    "EUR": 0.85,
    "GBP": 0.74,
    "JPY": 110.25,
    "CAD": 1.35,
    "AUD": 1.52,
    "CHF": 0.92,
    "CNY": 7.25,
    "INR": 83.12
  }
}
GET
/convert/latest_rates/{api_key}/{base_currency}/{target_currency}/{amount}

Converts a specific amount from the base_currency to the target_currency using the latest exchange rates. The amount must be a positive integer.

Parameters
Name Type Required Description
api_key string Yes Your unique API key
base_currency string Yes Currency code to convert from (e.g., USD)
target_currency string Yes Currency code to convert to (e.g., EUR)
amount number Yes Amount to convert
Response Example
{
  "meta": {
    "base_currency": "USD",
    "target_currency": "EUR",
    "amount": 100,
    "last_updated_at": "2024-10-24T18:00:00"
  },
  "data": {
    "conversion_rate": 0.85,
    "converted_amount": 85.0
  }
}

Error Responses

The API returns detailed error responses to help with debugging. All error responses follow this format:

Error Response Format
{
  "error": {
    "code": 401,
    "message": "Invalid API key"
  }
}

Note: All error responses follow this structured format with a numeric code and descriptive message field.

Status Code Error Type Description
400 Bad Request The request was malformed or contains invalid parameters
401 Unauthorized Missing or invalid API key
404 Not Found The requested resource could not be found
429 Too Many Requests Rate limit exceeded. Maximum 30 requests per minute per API key.
500 Server Error An error occurred while processing your request
503 Service Unavailable External currency API is unavailable and no fallback data exists

Graceful Fallbacks

If the external currency API is unavailable, the service will automatically return cached data with a warning:

Fallback Response Format
{
  "meta": {
    "base_currency": "USD",
    "last_updated_at": "2024-10-22T18:00:00"
  },
  "warning": {
    "message": "Using stale data as fallback. External API could not be reached.",
    "data_age": "2 days old"
  },
  "data": {
    "EUR": 0.85,
    "GBP": 0.74,
    "JPY": 110.25,
    "CAD": 1.35,
    "AUD": 1.52
  }
}
Note on Fallback Data

While using fallback data, the exchange rates may not reflect the current market conditions. The API will include a warning with the response indicating that the data is stale and its age.

Code Examples

Get latest exchange rates using the Fetch API:

JavaScript Example
// Fetch latest exchange rates
const apiKey = '{{api_key}}';
const baseCurrency = 'USD';
const url = `https://imhotepexchangeratesapi.pythonanywhere.com/latest_rates/${apiKey}/${baseCurrency}`;

fetch(url)
  .then(response => {
    if (!response.ok) {
      return response.json().then(err => {
        throw new Error(`API Error: ${err.error.message}`);
      });
    }
    return response.json();
  })
  .then(data => {
    console.log('Exchange rates:', data.data);
    console.log(`Last updated: ${data.meta.last_updated_at}`);
    
    // Check for stale data warning
    if (data.warning) {
      console.warn('Warning:', data.warning.message);
    }
    
    // Access specific rate
    const euroRate = data.data.EUR;
    console.log(`1 ${baseCurrency} = ${euroRate} EUR`);
  })
  .catch(error => {
    console.error('Error fetching exchange rates:', error);
  });

Get latest exchange rates using Python requests library:

Python Example
import requests

api_key = '{{api_key}}'
base_currency = 'USD'
url = f'https://imhotepexchangeratesapi.pythonanywhere.com/latest_rates/{api_key}/{base_currency}'

try:
    response = requests.get(url, timeout=10)
    response.raise_for_status()  # Raise exception for 4XX/5XX responses
    data = response.json()
    
    print(f"Exchange rates (base: {data['meta']['base_currency']})")
    print(f"Last updated: {data['meta']['last_updated_at']}")
    
    # Check for stale data warning
    if 'warning' in data:
        print(f"⚠️  Warning: {data['warning']['message']}")
        print(f"   Data age: {data['warning']['data_age']}")
    
    # Print rates for common currencies
    common_currencies = ['EUR', 'GBP', 'JPY', 'CAD', 'AUD', 'CHF', 'CNY']
    for currency in common_currencies:
        if currency in data['data']:
            rate = data['data'][currency]
            print(f"1 {base_currency} = {rate} {currency}")
            
except requests.exceptions.HTTPError as e:
    error_data = e.response.json() if e.response else {}
    print(f"HTTP Error {e.response.status_code}: {error_data.get('error', {}).get('message', str(e))}")
except requests.exceptions.RequestException as e:
    print(f"Request error: {e}")

Get latest exchange rates using Node.js with axios:

Node.js Example
const axios = require('axios');

const apiKey = '{{api_key}}';
const baseCurrency = 'USD';
const url = `https://imhotepexchangeratesapi.pythonanywhere.com/latest_rates/${apiKey}/${baseCurrency}`;

axios.get(url)
  .then(response => {
    const data = response.data;
    console.log('Exchange rates:', data.data);
    console.log(`Last updated: ${data.meta.last_updated_at}`);
    
    // Check for stale data warning
    if (data.warning) {
      console.warn('Warning:', data.warning.message);
      console.warn('Data age:', data.warning.data_age);
    }
    
    // Access specific rate
    const euroRate = data.data.EUR;
    console.log(`1 ${baseCurrency} = ${euroRate} EUR`);
  })
  .catch(error => {
    if (error.response) {
      console.error('API Error:', error.response.data.error.message);
    } else {
      console.error('Request Error:', error.message);
    }
  });
