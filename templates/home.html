Welcome to Imhotep Exchange Rate API

Your unique API key with the full URL is displayed below. Use it to access real-time currency exchange rates.

http://127.0.0.1:5000/latest_rates/{{api_key}}/USD

How to Use the API

Follow these steps to start using the Imhotep Exchange Rate API:

  1. Make sure you have your API key. You can copy it from the box above.
  2. To get the latest currency exchange rates, make a GET request to the following URL:
    https://yourapi.com/latest_rates/YOUR_API_KEY/USD
  3. You can change the base currency by replacing USD with any supported currency code, for example:
    https://yourapi.com/latest_rates/YOUR_API_KEY/EUR
  4. In the response, you'll receive real-time exchange rates relative to the base currency you specified.
