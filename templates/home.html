Sandbox Pre-filled Code References

We have compiled the documentation below with your credentials. Copy code examples directly into your project files.

GET

/latest_rates/{{api_key}}/{base_currency}

Retrieve active currency exchange rates relative to the requested currency parameter. 

// Fetch latest exchange rates with your credentials
const url = 'https://imhotepexchangeratesapi.pythonanywhere.com/latest_rates/{{api_key}}/USD';

fetch(url)
  .then(res => res.json())
  .then(data => {
    console.log('Exchange rates:', data.data);
  })
  .catch(err => console.error(err));
GET

/convert/latest_rates/{{api_key}}/{base_currency}/{target_currency}/{amount}

Convert currency metrics directly on our servers using the latest cached exchange rate arrays. 

// Convert currency amounts dynamically with your credentials
const url = 'https://imhotepexchangeratesapi.pythonanywhere.com/convert/latest_rates/{{api_key}}/USD/EUR/100';

fetch(url)
  .then(res => res.json())
  .then(data => {
    console.log(`Converted: ${data.data.converted_amount}`);
  })
  .catch(err => console.error(err));
